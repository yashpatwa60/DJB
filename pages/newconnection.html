Checklist for Applying New Water/Sewer Connection

Please go through checklist before start filling form

    Please refer below instructions/guidelines to fill the application form:

    Note:

    In case of applicant already having water connection and wish to apply sewer connection, the application through 'Add Services' tab to be applied.

    In case of applicant already having sewer connection and wish to apply water connection, the application through 'Add Services' tab to be applied.


    On New Connection application form below details need to be filled/entered/selected by applicant:
  • Applicant has to select ZRO location of their Residence constituency
  • Under Type of Request section - Applicant has to select one out of below mentioned radio buttons:
    1. Water & Sewerage Connection - Select this radio button if wish to apply for Water & Sewerage connection both
    2. Water Connection - Select this radio button in case where no Sewerage System has been laid by DJB
    3. Sewerage Connection - Select this radio button in case Water Connection is not required
    4. Regularization - Select this radio button in case water/sewer connection is already installed without prior sanction
  • Under Details of Applicant section - Applicant has to fill First Name, Middle Name, Last Name, Father/Husband Name. In case applicant belongs to Govt. Organization then Govt. Organization radio button is selected & then Name of Government Organization need to filled & scanned copy of Govt ID proof need to be uploaded.
  • Under Contact Details section - Email ID, Mobile No(Mandatory), Home Tel. No, Office No need to be filled by applicant.
  • Under DJB Employee section - If applicant is DJB employee then DJB Employee checkbox need to be checked & relevant details should be provided.
  • Under Property Address section - Applicant has to enter PIN Code & select Locality, Sub locality from dropdown populated on the basis of PIN Code & fill House No & other relevant details if applicable.
  • Under Property and water connection use details section - Applicant has to fill/select below details:
    1. Property Type: Applicant has to select one Property type out of the list. (Mandatory)
    2. Urban/Rural: Applicant has to select whether property belongs to Urban or Rural. (Mandatory)
    3. No of Floors: Applicant has to enter No of floors of Property.(To be fed by single connection case of JJ Residential Colony)
    4. Plot Area(in Sq m): Enter Plot Area in square Meters. (Mandatory)
    5. Built Up Area: Enter Plot Area in square Meters less than Plot Area. (Mandatory)
    6. Water Connection Type: Applicant has to select purpose of Water Connection Type to be applied out of Commercial/Industrial or Domestic. (Mandatory)
    7. Water connection used by (for identification/tagging-purpose): Applicant has to select Water Connection Type used by for identification/tagging purpose out of the list.(Mandatory)
  • Under Details of Licensed Plumber (for installation of connection proposed to be engaged by applicant) list available on website section - In accordance with section 35 of Delhi Water Board act, installation of connection shall be executed through licensed plumber approved by board.
  • Under Documents to be attached section - Applicant has to upload scanned copy of documents for below mentioned category:
    1. Proof of Identity - List of acceptable documents are available at top of page.
    2. Ownership Status - List of acceptable documents are available at top of page.
    3. Property Ownership Document - List of acceptable documents are available at top of page.
    4. Others - List of acceptable documents are available at top of page.
  • Under Declaration/Undertaking section - Applicant has to read Declaration and Undertaking carefully & enter Text from Captcha image displayed at left.
  • Under Submitted By section - Applicant has to Upload his/her Signature & check I Agree checkbox.
  • After submitting all above mentioned details, applicant has to click on Preview button.
  • Applicant can also Edit New Connection details as displayed in Preview form or can Submit the request if all entered details are correct.

  • Passport size photograph(File type - .jpg, .jpeg or .png of file size less than 150KB but the same is not mandatory)
  • ID proof of the Govt. organization(if applicant selects Govt. Organization checkbox - for taking connection on behalf of Govt. Organization)
  • Confirmation Letter of Licensed Plumber - In accordance with section 35 of Delhi Water Board act, installation of connection shall be executed only through licensed plumber approved by board.
  • Bank Details section - Applicant has to enter Name of the Bank, Name of the Branch, IFSC Code, Bank Account No. (Non-mandatory)
  • Rain Water Harvesting section - If applicant wish to apply for RWH Adequacy Certificate then Apply for RWH Adequacy Certificate radio button is to be selected & required data needs to be filled & saved. Also, Approved building sanction plan is to be uploaded which is mandatory in this case.
  • Proof Of Identity - One out of below documents need to be uploaded(Mandatory)
    1. Driving License, PAN Card, Passport, Ration card with Applicant's photo ,Photo Identity Card issued by any govt. agency, Unique ID card[UID/Aadhaar Card], Voter Identification Card
  • Self-Declaration Signed Document
    1. Document need to be downloaded, filled, signed & uploading the same in all cases other than having registered document of Ownership(Mandatory)
  • Property Ownership document - One out of below documents need to be uploaded(Mandatory):
    1. Regd Sale Deed - Below mentioned all relevant pages of Regd Sale Deed must be uploaded: First Page, Page mentioning details of both the parties. Registration details page, Pages mentioning Plot Size, Unit Size/Built up Area, Last page having Signature.
    2. Electricity Bills in the name of Applicant(any 3), Regd Perpetual Lease Deed, Regd Conveyance Deed, Regd GPA, Ntd. GPA with complete backchain, SPA with Regd/Ntd. Agreement to Sell and Will, Will/NOC of the owner in favour of applicant, Regd Partition Deed , Regd Family settlement, Regd Relinquished Deed, Allotment letter(pvt builder) with regd Sale Deed, Allotment letter(pvt builder) with Regd/Ntd. GPA, Allotment letter(pvt builder), Regd/Ntd. Agreement to Sell, Rented proprty-Lease/Rent Agreement, NOC, Ownership Proof, DDA Allotment/mutation Letter/Land, Development Letter, Interim Court order
  • Others(wherever applicable) - One out of below documents need to be uploaded:
    1. NOC from Owner (If applicant is other than owner along with document of owner), NOC from MCD (If Commercial property is present in Residential Area), NOC from DPCC (If Commercial property is present in Residential)
    2. In case of Bulk Connection the following documents are needed: Layout Plan, Key Plan, Floor Plan, Site Plan, Total Water Demand, Completion Plan, Hydraulic Calculation Plan,
    3. Rain Water Harvesting Certificate (Mandatory for plot size more than 100 sq mt & failure of which will levy penalty as per of rules in regular bills after moratorium period upto 30.06.2022)
  • Signature (File type - .jpg, .jpeg or .png of file size less than 20KB) (Mandatory)

Checklist for Applying RWH Certificate

ensure to carry following scanned copy of documents before start of filling online form for RWH Certificate:
  • Approved building sanction plan(File type - .jpg, .jpeg, .png or pdf of file size less than 2MB)
    • Please refer below instructions/guidelines to fill the application form:

  • Applicant has to enter Correspondence/Site Address of site where Rain Water Harvesting system is installed.
  • Under section Required Retention Capacity(in cum) - Applicant has to fill below mentioned details:
    1. Roof Top Area(In Sq m) Applicant has to fill Roof Top Area in square meters
    2. Paved/Pucca floor with non-pollution catchment area(In Sq m) Applicant has to fill Paved/Pucca floor with non-pollution catchment area in square meters
    3. Total Plot Area(In Sq m) - Applicant has to fill Total Plot Area in square meters.
  • Under section Existing Retention Capacity(in cum) Applicant has to fill below mentioned details:
    1. Size of Pit - Applicant has to select whether Size of Pit is Rectangular or Circular:

      If applicant selects Size of Pit as Rectangular then below details need to be filled: Length, Breadth & Depth in meter is entered accordingly which Total capacity in cum is calculated which is calculated by below formula: Total Capacity = Length * Breadth * Depth

      If applicant selects Size of Pit as Circular then below details need to be filled: Diameter (Dia) & Depth in meter is entered accordingly which Total capacity in cum is calculated by below formula: Total Capacity = 3.14 * (Diameter * Diameter * Depth) / 4
  • Under section Details of RWH Pit - Applicant has to select whether RWH Pit is Govt (Government) or Pvt (Private).
  • Under section Other parameters for RWH System - Applicant has to fill below mentioned details:
    1. Bypass/Overflow system exist - Applicant has to select Yes/No whether Bypass/Overflow system exists or not.
    2. System is Functional - Applicant has to select Yes/No whether system is functional or not.
    3. System is Adequate - Applicant has to select Yes/No whether system is adequate or not.
    4. Proper conveyance system exists - Applicant has to select Yes/No whether proper conveyance system exists or not.
  • Under section Filter Media Details - Applicant has to fill below mentioned details:
    1. Brick Aggregates 40 to 63 mm, size 0.5m thickness - Applicant has to select Yes/No in this field for the filter size.
    2. Charcoal/Activated carbon 0.1m thickness - Applicant has to select Yes/No whether Charcoal/Activated carbon is of 0.1m thickness or not.
    3. Sand (1.5mm - 2mm), size 0.2m thickness - Applicant has to select Yes/No in this field for the sand size.
    4. Geo textile membrane - Applicant has to select Yes/No whether Geo textile membrane exists or not.
  • After submitting all above mentioned details, applicant has to click on Save Details button.
